# periodic-table-vue

一个基于 Vue 3 和 Vite 构建的交互式元素周期表项目。

## 简介

- 支持中英文语言切换
- 支持按元素分类高亮显示
- 支持查看元素的原子结构可视化信息

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
